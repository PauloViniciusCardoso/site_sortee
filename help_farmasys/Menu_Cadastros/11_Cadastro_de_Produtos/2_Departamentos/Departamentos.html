Departamentos

O que são departamentos?

São divisões correspondentes a um ramo de produtos, serviços ou área.

+Qual o procedimento para cadastrar um departamento?

    No sistema:
  • Informe a descrição do departamento, vincule a um grupo de departamento, selecione a opção de comissão caso seja pago a esse departamento e informe o percentual de comissão pago e tecle ENTER até a próxima linha para salvar.
  • OBS.: Caso não tenha comissão, deixe o campo perc% com valor 0.00.

    • As informações sobre cada campo estão no final da página.

+Como alterar os dados de um departamento?

  • Selecione o departamento, faça as alterações e tecle ENTER até a próxima linha para salvar.

+É possível excluir um departamento?

  • Sim, mas deve-se atentar aos seguintes detalhes:
  • Se existirem produtos vinculados a esse departamento, a exclusão não será permitida.

+Descrição dos campos:



  • Código: Preenchido automaticamente pelo sistema.
  • Descrição: Informe a descrição do departamento.
  • Grupo: Informe o grupo de departamento.
  • Exemplo: Ótico (departamento) vinculado ao grupo de departamento 000 (medicamento).
  • Comissão: Selecione se os produtos vinculados a esse departamento tiverem a venda comissionada.
  • Perc%: Informe o percentual de comissão para o departamento.
