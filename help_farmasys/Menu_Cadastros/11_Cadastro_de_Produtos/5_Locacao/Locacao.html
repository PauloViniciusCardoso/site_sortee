Locação

O que são locações?

São subdivisões dos departamentos. Podem ser usadas para identificar onde os produtos estão alocados na farmácia. Aperte no topico que deseja para conhecer a funcionalidade de cada opção.

+Qual o procedimento para cadastrar uma locação?

    No sistema:
  • Informe a descrição da locação e tecle ENTER até a próxima linha para salvar.

    • OBS.: As informações sobre cada campo estão no final da página.

+Como alterar os dados de uma locação?

  • Selecione a locação, faça as alterações e tecle ENTER até a próxima linha para salvar.

+É possível excluir uma locação?

  • Sim, mas deve-se atentar aos seguintes detalhes:
  • Se existirem produtos vinculados a essa locação, a exclusão não será permitida.

+Descrição dos campos:

  • Código: Preenchido automaticamente pelo sistema.
  • Descrição: Informe a descrição da locação.
