Classificação Tributária

O que é classificação tributária?

São as alíquotas tributárias que incidem sobre as mercadorias para emissão de documentos de saída (cupom fiscal, nota fiscal eletrônica). Aperte no topico que deseja para conhecer a funcionalidade de cada opção.

+Qual o procedimento para cadastrar uma classificação tributária?

    O cadastro da classificação tributária é efetuado de acordo com a impressora fiscal (ECF).

    Para cadastrar, é necessário logar no Farmasys com o usuário administrador do sistema. Acesse o menu Utilitários >> Cadastro de Senhas.

    Clique duas vezes em "Liberar o acesso e classificação tributária dos produtos".

    Informe uma senha e clique em "Salvar".

    Informe o código de liberação e clique em "OK". Obs: O código de liberação é DD/MM/AA. Ex: 150914.

    Confirme a senha e clique em "OK".

    OBS.: Esse processo deverá ser executado pelo suporte do sistema.

+Como alterar os dados de uma classificação tributária?

    Aperte F5, informe a senha, selecione a classificação tributária, faça as alterações e tecle ENTER até a próxima linha.

+É possível excluir uma classificação tributária?

    Sim, mas deve-se atentar aos seguintes detalhes:

    Se existirem produtos vinculados a essa classificação tributária, a exclusão não será permitida.

+Descrição dos campos:

  • Código: Preenchido automaticamente pelo sistema.
  • Descrição: Informe a descrição da classificação tributária.
  • ECF: Conforme a posição cadastrada na leitura X.
  • Sub. trib: Selecione se a classificação for substituição tributária.
  • %ICMS: Informe a porcentagem de ICMS.
  • A tabela ao lado ICMS informa os valores em cada estado.
