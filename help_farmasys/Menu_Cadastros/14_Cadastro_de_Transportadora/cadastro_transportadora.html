Cadastro de Transportadoras

O que são transportadoras?

Empresa especializada no transporte de carga ou de passageiros.[cite: 5]

+Qual o procedimento para cadastrar uma transportadora?

    No sistema:
  • Informe os dados da empresa transportadora e tecle ENTER até a proxíma linha para salvar.[cite: 5]
  • OBS.: as informações sobre cada campo, estão no final da página.[cite: 5]

+Como alterar os dados de uma transportadora?

  • Selecione a transportadora, faça as alterações e tecle ENTER até a proxíma linha para salvar.[cite: 5]

+É possível excluir uma transportadora?

  • Sim, mas recomendamos não excluir a transportadora, pois esse cadastro é necessário para a emissão de nota fiscal eletrônica.[cite: 5]

+Descrição dos campos:

  • Código: Preechido automaticamente pelo sistema.[cite: 5]
  • Nome da transportadora: Informe o nome da empresa transportadora.[cite: 5]
  • Endereço: Informe o endereço da transportadora.[cite: 5]
  • Cidade: Informe a cidade.[cite: 5]
  • UF: Informe o estado.[cite: 5]
  • Telefone: Informe o telefone.[cite: 5]
  • Inscrição estadual: Informe a inscrição estadual.[cite: 5]
  • CNPJ: Informe o CNPJ.[cite: 5]
