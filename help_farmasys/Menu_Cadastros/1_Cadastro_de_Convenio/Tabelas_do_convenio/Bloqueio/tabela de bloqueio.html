Tabela de Bloqueio

O que é uma tabela de bloqueio?

É uma tabela criada para bloquear produtos no ato da venda, podendo ser classificados por: DEPARTAMENTOS, FAMÍLIA, LOCAÇÃO e LABORATÓRIOS.

+Procedimento para cadastrar uma tabela de bloqueio?

    No sistema:
    Clique no ícone ou pressione (F3):
    -> Para que o sistema gere um código sequencial (automático), note que os campos CÓDIGO e DATA serão preenchidos automaticamente.
    -> Você pode, se desejar, informar/digitar o seu código de controle. Caso este código não esteja utilizado, o sistema permitirá o cadastro.
    -> Selecione todos os campos pertinentes a serem bloqueados.

    OBS.: as informações sobre cada campo, estão no final da página.

+É possível alterar uma tabela de bloqueio?

    Sim, selecione a tabela em questão, informe os dados a serem alterados e pressione ENTER para salvar.

+É possível excluir uma tabela de bloqueio?

    Sim, mas deve-se atentar aos seguintes detalhes:
    Se a tabela estiver vinculada em algum convênio, a exclusão não será permitida.
