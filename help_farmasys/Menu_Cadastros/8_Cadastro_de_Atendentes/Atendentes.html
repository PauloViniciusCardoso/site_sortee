Cadastro de Atendentes

O que são atendentes?

São todas as pessoas que vão ter funções de atendimento na empresa. Aperte no topico que deseja para conhecer a funcionalidade de cada opção.

+Qual o procedimento para cadastrar atendentes?

    No sistema:
  • Informe a descrição do atendente (nome), a comissão e selecione se ele está ativo. O código deixe em branco, pois o sistema gera automaticamente. Tecle ENTER até a próxima linha para salvar.
  • OBS.: As informações sobre cada campo estão no final da página.

+Como alterar os dados do atendente?

  • Selecione o atendente, faça as alterações e tecle ENTER até a próxima linha para salvar.

+É possível excluir um atendente?

  • Sim, mas deve-se atentar aos seguintes detalhes:
  • Se existirem movimentações fiscais vinculadas a este atendente, a exclusão não será permitida.
  • Obs.: Se o atendente sair da empresa, desative-o no sistema.

+Descrição dos campos:

  • Código: Gerado automaticamente pelo sistema.
  • Descrição: Coloque o nome do atendente.
  • Comissão: Informe a porcentagem de comissão, caso não tenha coloque (0.00). Lembrando que essa porcentagem será geral sobre todos os produtos vendidos.
  • Ativo: Selecione se o atendente estiver ativo.
