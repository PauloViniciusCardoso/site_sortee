Venda para clientes cadastrados

Esta funcionalidade apresenta um histórico detalhado de vendas realizadas para clientes previamente cadastrados no sistema, incluindo todas as transações feitas para esses clientes.

+Guia Utilização do Filtro

  • Seleção do Período: Informe o período desejado para a busca.
  • Opções de Detalhe: Se necessário, especifique mais opções para uma busca mais precisa.
  • Processar: Clique em "Processar" para iniciar a busca.

    O filtro buscará todas as informações de compras realizadas por clientes cadastrados no período informado. Os resultados serão apresentados na aba "Compras". Ao clicar no símbolo ao lado de cada compra, é possível ver detalhes dos produtos comprados pelo cliente.

    Ao clicar em "Análise", serão apresentadas informações detalhadas do cliente selecionado.

+Guia Estatística

  • Limite de Crédito:Limite de crédito liberado para o cliente.
  • Saldo Devedor Total:Valor total que o cliente deve.
  • Saldo Devedor Atrasado:Contas vencidas.
  • Saldo em Crédito:Crédito disponível.
  • Dia de Pagamento:Dia programado para pagamento, se houver.
  • Desconto no Cadastro:Desconto específico para o cliente.
  • Ticket Médio (90 dias):Média das vendas em valor para o cliente nos últimos 90 dias.
  • Média de Compra Mensal (90 dias):Média mensal em valor das compras do cliente nos últimos 90 dias.
  • Média de Devolução Mensal (90 dias):Média mensal em valor das vendas devolvidas e canceladas pelo cliente nos últimos 90 dias.
  • Média de Atraso no Pagamento (dias):Média de dias de atraso no pagamento.

    • Ao selecionar um produto, são apresentadas as últimas 50 operações detalhadas daquele produto.

+Guias Adicionais

  • Compras:Detalhes das vendas realizadas para o cliente.
  • Cheque:Histórico de vendas registradas em cheque.
  • Contas a Receber:Contas abertas e pendentes de pagamento.
  • Cartões:Informações de vendas realizadas com cartão.
  • LOG's:Operações realizadas no cadastro do cliente.
  • Créditos:Histórico de entradas e saídas de crédito.
